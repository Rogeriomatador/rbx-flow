RBX Flow · Roblox Mobile Assistant

Um assistente seguro para Roblox Mobile.

O RBX Flow é um launcher/assistente criado para organizar o perfil do jogador, avatar, favoritos, histórico, atalhos úteis e uma bolinha flutuante para acompanhamento durante o uso do celular.

O app não pede senha do Roblox, não usa cookie de sessão, não injeta código e não modifica o aplicativo oficial do Roblox. A conexão de conta é feita apenas por autorização oficial OAuth, quando disponível.

Perfil conectadoAvatar, nome, User ID e dados autorizados pela conta Roblox.
Bolinha flutuanteAtalhos, guia visual, último jogo e ferramentas rápidas.
SeguroSem senha, sem cookie .ROBLOSECURITY e sem acesso escondido.
