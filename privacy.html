Política de Privacidade

Esta página explica quais dados o RBX Flow pode usar, por que eles são usados e como a conexão com Roblox deve funcionar.

Última atualização: 30 de junho de 2026.

1. O que é o RBX Flow

O RBX Flow é um assistente/launcher independente para Roblox Mobile. Ele pode organizar perfil, avatar, grupos, experiências, ativos, favoritos, histórico e ferramentas visuais.

2. Dados básicos

O app pode salvar localmente username, display name, User ID, avatar público, Place IDs, favoritos, histórico, preferências e status de conexão.

3. OAuth Roblox

A conexão com Roblox deve ocorrer por OAuth oficial. O usuário autoriza as permissões na própria tela do Roblox antes do app acessar dados ou recursos.

4. Permissões avançadas

Importante: se o usuário autorizar recursos avançados, o app poderá acessar informações e ferramentas adicionais permitidas pelo Roblox, como comunidades, ativos, passes, produtos e experiências.

5. O que o app não pede

  • Não pede senha do Roblox.
  • Não usa credenciais ocultas.
  • Não acessa a conta sem autorização.
  • Não modifica o aplicativo oficial do Roblox.

6. Uso dos dados

Os dados são usados para exibir perfil, personalizar a interface, abrir jogos, mostrar recursos autorizados, salvar preferências e melhorar a experiência dentro do RBX Flow.

7. Armazenamento

Dados do app podem ser armazenados no dispositivo. Tokens, quando usados, devem ser protegidos por armazenamento seguro no aparelho.

8. Compartilhamento

O RBX Flow não vende dados pessoais. Dados só devem ser enviados para serviços necessários, como APIs oficiais do Roblox, quando autorizado.
