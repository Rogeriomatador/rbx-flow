Política de Privacidade do RBX Flow

Última atualização: 30 de junho de 2026.

1. O que é o RBX Flow

O RBX Flow é um launcher/assistente seguro para Roblox Mobile. Ele organiza atalhos, perfil, favoritos, histórico e ferramentas de apoio visual.

2. Dados coletados

O RBX Flow pode salvar localmente no dispositivo informações como username, display name, User ID, avatar público, Place IDs favoritos, histórico de jogos e preferências do app.

3. Conexão com Roblox

Quando houver conexão com Roblox, ela deve ocorrer por autorização oficial OAuth. O RBX Flow não solicita senha do Roblox, não solicita cookie .ROBLOSECURITY e não acessa a conta escondido.

4. Uso dos dados

Os dados são usados para exibir perfil, personalizar a experiência, salvar favoritos, abrir jogos e mostrar informações autorizadas pelo usuário.

5. Armazenamento

As informações do app podem ser armazenadas localmente no dispositivo. Tokens de autenticação, quando usados, devem ser armazenados de forma segura no aparelho.

6. Compartilhamento

O RBX Flow não vende dados pessoais. Dados só são enviados para serviços necessários ao funcionamento do app, como APIs oficiais do Roblox, quando o usuário autoriza.

7. Segurança

O app não injeta código no Roblox, não modifica o aplicativo oficial e não tenta burlar sistemas de segurança.

Contato do projeto: GitHub do RBX Flow.
